WhiteBoard Help

Page No.

It shows the current page number against total pages.

Color

Select pen color.

Eraser E

Check the box to erase. Note: It will not work for transparent page.
E will toggle between eraser and selected color.

Size

Set the pen size.

Tune

Tune provides you the flow in which you want to write. For example, Tune=1 will give you dotted lines while Tune=10000 will give you a straight line.
Based on your use case choose the correct Tune value
20 is suitable for slow writing, if you will scribble fast it will give you dotted lines at some places.
100 is recommended for fast writing and drawing. It will give you dotted lines only when you will scribble very much fast.
Note:1 is its least value while 10000 is its highest value.
The higher is its value the faster you have to scribble to make dotted lines and response may get slow.

Clear Canvas Backspace

It will clear the canvas.

New Page N

It will create new page.

Page Up Arrow Up

It will go one page behind.

Page Down Arrow Down

It will go one page forward.

Transparent Page T

It will open page with transparent background.

Delete Page Delete

It will delete current page.

Save Page Shift+S

It will save only current page in the device.

Save All Shift+A

It will save all pages as png in the device.
